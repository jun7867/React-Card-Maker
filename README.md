## 프로젝트 목적

-   React hook 연습
-   firebase api를 사용하여 React와 연동해보기
-   cloudinary api 사용하여 이미지 파일 관리 학습
-   React router 학습
-   MVC 패턴 적용 연습

## 사용 기술

<span id="use-tech" />
React Hook, Firebase , MVC Pattern, Cloudinary (image upload), dependency injection, memo
